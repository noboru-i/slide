簡単に捨てられる環境が欲しい

Vagrant

http://www.vagrantup.com/

Ubuntuの仮想マシンを
立ち上げる


$ vagrant box add base http://files.vagrantup.com/lucid32.box
$ vagrant init
$ vagrant up

いらなくなったら


$ vagrant destroy

http://www.opscode.com/chef/

仮想マシンの立ち上げ

同じようなインフラ構成のプロジェクトで流用できる

iOSのライブラリを管理したい

CocoaPods

http://cocoapods.org/

JSONKitは1.4以降、Reachabilityは3.0.0以降を使いたい


$ vim Podfile
  platform :ios
  pod 'JSONKit',      '~> 1.4'
  pod 'Reachability', '~> 3.0.0'
$ pod install

CSSを綺麗に書きたい

Sass

http://sass-lang.com/

出来ること

  • 変数の定義
  • セレクタのネスト
  • 継承

$vmargin: 2em;
table.hl {
  margin: $vmargin 0;
  td.ln {
    text-align: right;
  }
}

が


table.hl {
  margin: 2em 0;
}
table.hl td.ln {
  text-align: right;
}

に変換されます

SCSS(Sassy CSS)という記法です。SASS記法もあります。
